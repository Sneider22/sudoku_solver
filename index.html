Reglas del Sudoku

El objetivo es rellenar el tablero de forma que cada fila, cada columna y cada subcuadrícula contengan todos los símbolos permitidos sin repetir.

  • 6x6: símbolos 1–6. Subcuadrículas de 2x3.
  • 9x9: símbolos 1–9. Subcuadrículas de 3x3.
  • 16x16: símbolos 0–9 y A–F. Subcuadrículas de 4x4.

Puedes introducir tu propio Sudoku (por ejemplo, del periódico). Usa Validar para comprobar si las entradas actuales cumplen las reglas y Resolver para que el sistema complete automáticamente una solución si existe.
